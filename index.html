Mars Rover Motherboard

Printed Circuit Board ~ Software

I designed and manufactured the first revision of the main control printed circuit board for the Missouri University of Science & Technology's Mars Rover Design Team. The board contained:

  • ARM M4 processor
  • AVR co-processor
  • GPS receiver
  • Multiplexed RS-485 transceivers for communication to the rest of the electronic subsystems

I also authored software in C that received commands from a basestation, aggregated telemetry from rover subsystems, and collected position data from the GPS receiver.

Mining User Similarity in Geo-Social Networks

Software

The pervasiveness of location­acquisition technologies (e.g., GPS) enables people to conveniently record the location history described by spatio­temporal trajectories. With plenty of location data, valuable information can be mined and applied to practice such as friends recommendation from trajectory similarity. In this project, we implemented an algorithm to calculate the spatial similarity of two trajectories, which is the basis to mine more semantic information of users.

8051 Mp3 Player

Breadboard ~ Software

Created mp3 player on a breadboard. Implemented a timer based state machine for an 8051 microcontroller to accept user song selection, read mp3 files using SPI interface from SD card, operate mp3 hardware decoder using I2C protocol, and display song information on LCD.
